      <p>
       <b>
        Screenwriting software
       </b>
       are
       <a class="mw-redirect" href="/wiki/Word_processors" title="Word processors">
        word processors
       </a>
       specialized to the task of writing
       <a href="/wiki/Screenplay" title="Screenplay">
        screenplays
       </a>
       .
      </p>

      </p>

      <h2>
       <span class="mw-headline" id="Overview">
        Overview
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Screenwriting_software&amp;action=edit&amp;section=1" title="Edit section: Overview">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <h3>
       <span class="mw-headline" id="Features">
        Features
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Screenwriting_software&amp;action=edit&amp;section=2" title="Edit section: Features">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       While add-ins and macros for word processors, such as
       <a class="new" href="/w/index.php?title=Script_Wizard&amp;action=edit&amp;redlink=1" title="Script Wizard (page does not exist)">
        Script Wizard
       </a>
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       for
       <a href="/wiki/Microsoft_Word" title="Microsoft Word">
        Microsoft Word
       </a>
       , can be used to write screenplays, the need for dedicated screenwriting programs arises from the presence of certain peculiarities in standard
       <a href="/wiki/Screenplay#Format_and_style" title="Screenplay">
        screenplay format
       </a>
       which are not handled well by generic word processors such as
       <a href="/wiki/Screenwriting#Page_breaks" title="Screenwriting">
        page-break constraints
       </a>
       imposed by standard screenplay format. Screenplay software often provides specialized shortcuts for quickly typing character names or
       <a href="/wiki/Screenplay#Format_and_style" title="Screenplay">
        slug lines
       </a>
       , collaborating with other writers, adding production notes, character notes, easy outlining, scene reordering, and other tools to facilitate the creative process.
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
       Other features often required when writing
       <a href="/wiki/Shooting_script" title="Shooting script">
        shooting scripts
       </a>
       include page-locking, scene numbering, revision-tracking, and production-related reports (such as which characters appear in a given scene or which scenes are set during the day or night).
      </p>
      <p>
       Some screenwriting applications, such as
       <a href="/wiki/Celtx" title="Celtx">
        Celtx
       </a>
       and
       <a href="/wiki/Sophocles_(software)" title="Sophocles (software)">
        Sophocles
       </a>
       , also incorporate
       <a href="/wiki/Shooting_schedule" title="Shooting schedule">
        production scheduling
       </a>
       and
       <a href="/wiki/Film_budgeting" title="Film budgeting">
        budgeting
       </a>
       features.
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
      </p>
      <p>
       Another class of screenwriting software includes those that, rather than act as specialized
       <a class="mw-redirect" href="/wiki/Word_processors" title="Word processors">
        word processors
       </a>
       , attempt to direct the writer's storytelling process by utilizing
       <a href="/wiki/Narratology" title="Narratology">
        story theory
       </a>
       to facilitate the planning of a screenplay. Examples of this type of program includes
       <a href="/wiki/Dramatica_(software)" title="Dramatica (software)">
        Dramatica
       </a>
       and
       <a class="new" href="/w/index.php?title=Index_Card_(software)&amp;action=edit&amp;redlink=1" title="Index Card (software) (page does not exist)">
        Index Card
       </a>
       .
      </p>
      <h3>
       <span class="mw-headline" id="Platforms">
        Platforms
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Screenwriting_software&amp;action=edit&amp;section=3" title="Edit section: Platforms">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       Screenwriting programs exist for all varieties of platforms and environments including traditional standalone desktop applications that run directly on a personal computer,
       <a class="mw-redirect" href="/wiki/Web_applications" title="Web applications">
        web applications
       </a>
       that run solely within a browser, and mobile apps that run on phones, tablets, and other portable devices.
      </p>
      <h3>
       <span class="mw-headline" id="Plain_text_markup">
        Plain text markup
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Screenwriting_software&amp;action=edit&amp;section=4" title="Edit section: Plain text markup">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       The
       <a href="/wiki/Fountain_(markup_language)" title="Fountain (markup language)">
        Fountain
       </a>
       markup syntax, co-developed by screenwriter
       <a href="/wiki/John_August" title="John August">
        John August
       </a>
       , facilitates formatting screenplays directly from plain text, be it in dedicated writing software, email programs, or text generated through
       <a href="/wiki/Optical_character_recognition" title="Optical character recognition">
        OCR
       </a>
       .
       <sup class="reference" id="cite_ref-4">
        <a href="#cite_note-4">
         [4]
        </a>
       </sup>
       There is also a formatting package for
       <a href="/wiki/LaTeX" title="LaTeX">
        LaTeX
       </a>
       called screenplay.
      </p>
      <h2>
       <span class="mw-headline" id="History">
        History
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Screenwriting_software&amp;action=edit&amp;section=5" title="Edit section: History">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The first screenwriting software was a standalone script formatter, Scriptor, from Screenplay Systems
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (January 2018)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
       . It took a text file generated by a word processor and inserted the proper page break tags. When used in conjunction with a
       <a class="mw-redirect" href="/wiki/Terminate_and_Stay_Resident" title="Terminate and Stay Resident">
        TSR
       </a>
       program such as
       <a href="/wiki/SmartKey" title="SmartKey">
        SmartKey
       </a>
       or ProKey—keyboard utilities that assigned a sequence of commands to keystroke combinations—the "dot commands" that Scriptor required could be inserted semi-automatically. Additionally, keyboard macros could be programmed to properly indent and enter abbreviations—allowing a user to customize the working of the word processor.
      </p>
      <p>
       SmartKey was popular with screenwriters from 1982–1987, after which word processing programs had their own macro features.
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (January 2018)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
      </p>
      <p>
       An update to Scriptor understood the style sheets used in Microsoft Word for DOS. And so the need for key macro programs was lessened. Scriptor's limitation was that once formatted it was difficult to re-import the resulting text back into a word processor for further editing.
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (January 2018)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
      </p>
      <p>
       The next generation of screenplay software hooked into Microsoft Word. Warren Script Application was initially released as a set of style sheets for Word for DOS. It was updated for Word for Windows circa 1988. gScript, a shareware script formatter/template, was released via CompuServe in 1989. It was included on the disk accompanying the book
       <i>
        Take Word for Windows to the Edge
       </i>
       , published by Ziff-Davis in 1993. It was subsequently updated and released commercially as ScriptWright.
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (January 2018)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
      </p>
      <p>
       This third generation of screenplay software consists of the standalone script writing programs such as Movie Magic Screenwriter, Final Draft, and Cinovation's Scriptware.
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (January 2018)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
      </p>
      <p>
       The latest generation adds online storage and collaboration and integrates with apps on mobile devices. Many software also integrate outlining tools as well as other creative support, and tools to further integrate the production process.
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (January 2018)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="List_of_software">
        List of software
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Screenwriting_software&amp;action=edit&amp;section=6" title="Edit section: List of software">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <ul>
       <li>
        <a href="/wiki/Celtx" title="Celtx">
         Celtx
        </a>
        - Free to register, but requires paid subscription for the full collaborative version. Browser-based, with apps for
        <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">
         Mac OS X
        </a>
        ,
        <a href="/wiki/IOS" title="IOS">
         IOS
        </a>
        , and
        <a href="/wiki/Android_(operating_system)" title="Android (operating system)">
         Android
        </a>
        . Includes a comic book script-writing mode.
        <sup class="reference" id="cite_ref-:0_5-0">
         <a href="#cite_note-:0-5">
          [5]
         </a>
        </sup>
       </li>
       <li>
        <a href="/wiki/Dramatica_(software)" title="Dramatica (software)">
         Dramatica
        </a>
        - Name of a theory and software suite created as part of a project by Chris Huntley and Melanie Anne Phillips.
       </li>
       <li>
        <a class="new" href="/w/index.php?title=DubScript_Screenplay_Writer&amp;action=edit&amp;redlink=1" title="DubScript Screenplay Writer (page does not exist)">
         DubScript Screenplay Writer
        </a>
        - A professional screenwriting software for
        <a href="/wiki/Android_(operating_system)" title="Android (operating system)">
         Android
        </a>
        with features such as auto-formatting and Google Docs fixing. A free version is available with a watermark header on printed output/PDF and optional subscription. DubScript also outputs to most major screenwriting formats.
        <sup class="reference" id="cite_ref-:0_5-1">
         <a href="#cite_note-:0-5">
          [5]
         </a>
        </sup>
       </li>
       <li>
        <a href="/wiki/Fade_In_(software)" title="Fade In (software)">
         Fade In Professional Screenwriting Software
        </a>
        - Professional screenwriting software for Mac OS X, Microsoft Windows and Linux, with mobile versions for iPhone, iPad and Android. Imports and exports Final Draft as well as most other major screenwriting software formats.
       </li>
       <li>
        <a href="/wiki/Final_Draft_(software)" title="Final Draft (software)">
         Final Draft
        </a>
        - Professional screenwriting software for
        <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">
         Mac OS X
        </a>
        and
        <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">
         Microsoft Windows
        </a>
        .
       </li>
       <li>
        <a class="new" href="/w/index.php?title=JotterPad_Pro&amp;action=edit&amp;redlink=1" title="JotterPad Pro (page does not exist)">
         JotterPad Pro
        </a>
        - A powerful professional screenwriting app for
        <a class="external text" href="https://play.google.com/store/apps/details?id=com.jotterpad.x&amp;hl=en" rel="nofollow">
         Android
        </a>
        . Includes paid
        <a href="/wiki/Fountain_(markup_language)" title="Fountain (markup language)">
         Fountain
        </a>
        support, export to
        <a href="/wiki/Final_Draft_(software)" title="Final Draft (software)">
         Final Draft
        </a>
        and print formatted script to PDF.
       </li>
       <li>
        <a class="external text" href="https://kitscenarist.ru/en/" rel="nofollow">
         KIT Scenarist
        </a>
        - A free and
        <a class="mw-redirect" href="/wiki/Open_source" title="Open source">
         open source
        </a>
        screenwriting software for
        <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">
         Mac OS X
        </a>
        ,
        <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">
         Microsoft Windows
        </a>
        and
        <a href="/wiki/Linux" title="Linux">
         Linux
        </a>
        , which includes story development tools, corkboard, screenplay editor and statistics module. Includes paid service for
        <a href="/wiki/Collaborative_real-time_editor" title="Collaborative real-time editor">
         real-time collaboration
        </a>
        and versions for
        <a href="/wiki/Android_(operating_system)" title="Android (operating system)">
         Android
        </a>
        and
        <a href="/wiki/IOS" title="IOS">
         iOS
        </a>
        .
       </li>
       <li>
        <a class="mw-redirect" href="/wiki/Montage_software" title="Montage software">
         Montage software
        </a>
        - Screenwriting software for Mac only.
       </li>
       <li>
        <a href="/wiki/Movie_Magic_Screenwriter" title="Movie Magic Screenwriter">
         Movie Magic Screenwriter
        </a>
        - Integrated professional screenwriting software. Available on Mac and Microsoft Windows.
       </li>
       <li>
        <a class="mw-redirect" href="/wiki/Page_2_stage" title="Page 2 stage">
         Page 2 stage
        </a>
        - Free open-source software, designed specifically for Windows. Available in a variety of languages.
       </li>
       <li>
        <a href="/wiki/Practical_Scriptwriter" title="Practical Scriptwriter">
         Practical Scriptwriter
        </a>
        - Professional script writing software for Windows. Formats cover Film, Stage, Radio and TV.
       </li>
       <li>
        <a class="new" href="/w/index.php?title=Script_Studio&amp;action=edit&amp;redlink=1" title="Script Studio (page does not exist)">
         Script Studio
        </a>
        (formerly
        <a href="/wiki/Movie_Outline" title="Movie Outline">
         Movie Outline
        </a>
        ) - Creative writing software for Mac and Windows designed by produced screenwriter for writing and professionally formatting movie scripts, teleplays, stage plays and novels. Includes story and character development features.
       </li>
       <li>
        <a class="new" href="/w/index.php?title=Scripts_Pro&amp;action=edit&amp;redlink=1" title="Scripts Pro (page does not exist)">
         Scripts Pro
        </a>
        - iOS-based app which allows a writer to edit scripts in text file format via Dropbox and iTunes transfers.
        <sup class="reference" id="cite_ref-:0_5-2">
         <a href="#cite_note-:0-5">
          [5]
         </a>
        </sup>
       </li>
       <li>
        <a href="/wiki/Scrivener_(software)" title="Scrivener (software)">
         Scrivener (software)
        </a>
        - A powerful content-generation tool that helps writes concentrate on composing and structuring long and difficult documents for
        <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">

        </a>
        and
        <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">

        <a href="/wiki/Linux" title="Linux">
         Linux
        </a>
        and
        <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">
         Microsoft Windows
        </a>
        .
       </li>
       <li>
        <a class="external text" href="https://writerduet.com" rel="nofollow">
         WriterDuet
        </a>
        - Real-time collaborative, online-offline professional screenwriting software.
        <sup class="reference" id="cite_ref-6">
         <a href="#cite_note-6">
          [6]
         </a>
        </sup>
       </li>
       <li>
        <a class="external text" href="http://www.nongnu.org/screenwriter/" rel="nofollow">

        </a>

        </a>
        files.
