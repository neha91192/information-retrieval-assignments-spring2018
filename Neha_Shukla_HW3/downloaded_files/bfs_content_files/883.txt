   <h1 class="firstHeading" id="firstHeading" lang="en">
    Production board
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div class="noprint" id="siteSub">
     From Wikipedia, the free encyclopedia
    </div>
    <div id="contentSub">
     <span class="mw-redirectedfrom">
      (Redirected from

       Production strip
      </a>
      )
     </span>
    </div>

    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <div class="mw-parser-output">
      <p>
       A traditional
       <b>
        production board
       </b>
       ,
       <b>
        stripboard
       </b>
       , or
       <b>
        production strip
       </b>
       is a
       <a href="/wiki/Filmmaking" title="Filmmaking">
        filmmaking
       </a>
       term for a cardboard or wooden chart displaying color-coded strips of paper, each containing information about a scene in the film's
       <a href="/wiki/Shooting_script" title="Shooting script">
        shooting script
       </a>
       . The strips can then be rearranged and laid out sequentially to represent the order one wants to film in, providing a schedule that can be used to plan the production. This is done because most films are shot "out of sequence," meaning that they do not necessarily begin with the first scene and end with the last. For logistical purposes, scenes are often grouped by
       <a class="mw-redirect" href="/wiki/Cast_member" title="Cast member">
        talent
       </a>
       or
       <a href="/wiki/Filming_location" title="Filming location">
        location
       </a>
       and are arranged to accommodate the schedules of
       <a class="mw-redirect" href="/wiki/Cast_member" title="Cast member">
        cast
       </a>
       and
       <a href="/wiki/Film_crew" title="Film crew">
        crew
       </a>
       . A production board is not to be confused with a
       <a href="/wiki/Stripboard" title="Stripboard">
        Stripboard
       </a>
       used for electronics prototyping.
      </p>
      <p>
       A modern version of a strip board will commonly be printed using dedicated computer software, such as MovieMagic Scheduling,
       <a href="/wiki/Celtx" title="Celtx">
        Celtx
       </a>
       , or
       <a href="/wiki/Scenechronize" title="Scenechronize">
        Scenechronize
       </a>
       , or by customizing general purpose software such as
       <a class="mw-redirect" href="/wiki/OpenOffice.org_Calc" title="OpenOffice.org Calc">
        OpenOffice.org Calc
       </a>
       or
       <a href="/wiki/Microsoft_Excel" title="Microsoft Excel">
        Microsoft Excel
       </a>
       .
      </p>
      <p>
      </p>

      <h2>
       <span class="mw-headline" id="Common_Contents">
        Common Contents
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Production_board&amp;action=edit&amp;section=1" title="Edit section: Common Contents">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Information on the strips can include
      </p>
      <ul>
       <li>
        The scene number
       </li>
       <li>
        The
        <a class="mw-redirect" href="/wiki/Time_of_day#Colloquial" title="Time of day">
         day
        </a>
        (Sunrise/Morning/Noon/Afternoon/Evening/Sunset/Night)
       </li>
       <li>
        The number of pages in that scene
        <ul>
         <li>
          This is commonly counted in eighths of a page.
          <sup class="reference" id="cite_ref-1">
           <a href="#cite_note-1">
            [1]
           </a>
          </sup>
         </li>
        </ul>
       </li>
       <li>
        The set that is described in the script
       </li>
       <li>
        The actual
        <a href="/wiki/Filming_location" title="Filming location">
         location
        </a>
        that will be filmed
       </li>
       <li>
        The characters in that scene
       </li>
       <li>
        Miscellaneous notes on the production
       </li>
      </ul>
