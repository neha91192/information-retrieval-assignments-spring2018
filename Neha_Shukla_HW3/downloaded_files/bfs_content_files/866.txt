      <p>
       The
       <b>
        Virtual International Authority File
       </b>
       (
       <b>
        VIAF
       </b>
       ) is an international
       <a class="mw-redirect" href="/wiki/Authority_file" title="Authority file">
        authority file
       </a>
       . It is a joint project of several national libraries and operated by the
       <a href="/wiki/OCLC" title="OCLC">
        Online Computer Library Center
       </a>
       (OCLC).
       <sup class="reference" id="cite_ref-LibraryJournal-VIAFtoOCLC_1-0">
        <a href="#cite_note-LibraryJournal-VIAFtoOCLC-1">
         [1]
        </a>
       </sup>
      </p>
      <p>
      </p>

      <h2>
       <span class="mw-headline" id="History">
        History
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Virtual_International_Authority_File&amp;action=edit&amp;section=1" title="Edit section: History">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Discussion about having a common international authority started in the late 1990s. After a series of failed attempts to come up with a unique common authority file, the new idea was to link existing national authorities. This would present all the benefits of a common file without requiring a large investment of time and expense in the process.
       <sup class="reference" id="cite_ref-IFLA-VIAF-Origins-2016_2-0">
        <a href="#cite_note-IFLA-VIAF-Origins-2016-2">
         [2]
        </a>
       </sup>
      </p>
      <p>
       The project was initiated by the US
       <a href="/wiki/Library_of_Congress" title="Library of Congress">
        Library of Congress
       </a>
       (LC), the
       <a href="/wiki/German_National_Library" title="German National Library">
        German National Library
       </a>
       (DNB) and the OCLC on August 6, 2003. The
       <a href="/wiki/Biblioth%C3%A8que_nationale_de_France" title="Bibliothèque nationale de France">
        Bibliothèque nationale de France
       </a>
       (BnF) joined the project on October 5, 2007.
      </p>
      <p>
       The project transitioned to being a service of the OCLC on April 4, 2012.
       <sup class="reference" id="cite_ref-OCLC-VIAF-Service-2012_3-0">
        <a href="#cite_note-OCLC-VIAF-Service-2012-3">
         [3]
        </a>
       </sup>
      </p>
      <p>
       The aim is to link the national authority files (such as the German
       <a href="/wiki/Name_Authority_File" title="Name Authority File">
        Name Authority File
       </a>
       ) to a single virtual authority file. In this file, identical records from the different data sets are linked together. A VIAF record receives a standard data number, contains the primary "see" and "see also" records from the original records, and refers to the original authority records. The data are made available online and are available for research and data exchange and sharing. Reciprocal updating uses the
       <a href="/wiki/Open_Archives_Initiative" title="Open Archives Initiative">
        Open Archives Initiative
       </a>
       <a href="/wiki/Protocol_for_Metadata_Harvesting" title="Protocol for Metadata Harvesting">
        Protocol for Metadata Harvesting
       </a>
       (OAI-PMH) protocol.
      </p>
      <p>
       The file numbers are also being added to
       <a href="/wiki/Wikipedia" title="Wikipedia">
        Wikipedia
       </a>
       biographical articles and are incorporated into
       <a href="/wiki/Wikidata" title="Wikidata">
        Wikidata
       </a>
       .
       <sup class="reference" id="cite_ref-OCLC-VIAFbot-2012_4-0">
        <a href="#cite_note-OCLC-VIAFbot-2012-4">
         [4]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-Code4Lib-VIAFbot-2013_5-0">
        <a href="#cite_note-Code4Lib-VIAFbot-2013-5">
         [5]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="VIAF_clusters">
        VIAF clusters
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Virtual_International_Authority_File&amp;action=edit&amp;section=2" title="Edit section: VIAF clusters">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       VIAF's clustering algorithm is run every month. As more data are added from participating libraries, clusters of authority records may coalesce or split, leading to some fluctuation in the VIAF identifier of certain authority records.
       <sup class="reference" id="cite_ref-DLib-VIAF-Ambiguity-2014_6-0">
        <a href="#cite_note-DLib-VIAF-Ambiguity-2014-6">
         [6]
        </a>
       </sup>
      </p>
