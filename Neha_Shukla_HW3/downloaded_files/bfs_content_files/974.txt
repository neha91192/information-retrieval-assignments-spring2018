   <h1 class="firstHeading" id="firstHeading" lang="en">
    Script coordinator
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div class="noprint" id="siteSub">
     From Wikipedia, the free encyclopedia
    </div>
    <div id="contentSub">
    </div>
    <div class="mw-jump" id="jump-to-nav">

     <a href="#mw-head">

     </a>

    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <div class="mw-parser-output">
      <p>
       A
       <b>
        script coordinator
       </b>
       is a role in the production of a
       <a href="/wiki/Film" title="Film">
        film
       </a>
       or
       <a class="mw-redirect" href="/wiki/Television_program" title="Television program">
        television series
       </a>
       . The script coordinator is responsible for producing each
       <a href="/wiki/Draft_document" title="Draft document">
        draft
       </a>
       of the
       <a href="/wiki/Screenplay" title="Screenplay">
        script
       </a>
       and annotating it for ease of use for the production team.
      </p>
      <p>
       On a television series the script coordinator is responsible for liaising between the writing and
       <a class="mw-redirect" href="/wiki/Film_production#Production" title="Film production">
        production
       </a>
       departments. The writers deliver the first
       <a href="/wiki/Draft_document" title="Draft document">
        draft
       </a>
       of a
       <a href="/wiki/Screenplay" title="Screenplay">
        script
       </a>
       to the script coordinator who prepares it for the production team and handles any clearance issues. Clearance refers to the need to check the
       <a href="/wiki/Screenplay" title="Screenplay">
        script
       </a>
       for potential legal problems and ensure that all names are cleared by the legal department.
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
      </p>
      <p>
       The script coordinator must check the
       <a href="/wiki/Draft_document" title="Draft document">
        draft
       </a>
       for proper formatting, spelling, punctuation and continuity before releasing the
       <a href="/wiki/Draft_document" title="Draft document">
        draft
       </a>
       . Once a
       <a href="/wiki/Screenplay" title="Screenplay">
        script
       </a>
       has been released, other members of the production team offer notes to the writers necessitating revisions to the
       <a href="/wiki/Screenplay" title="Screenplay">
        script
       </a>
       . The process of how widely each
       <a href="/wiki/Draft_document" title="Draft document">
        draft
       </a>
       is released (e.g. just to the writers, then to the writers and producers and then to the network) varies from show to show.
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
      </p>
      <p>
       With each round of revisions the script coordinator must ensure that the changes are clearly marked, re-issue the
       <a href="/wiki/Screenplay" title="Screenplay">
        script
       </a>
       and again check the
       <a href="/wiki/Screenplay" title="Screenplay">
        script
       </a>
       for errors. Many script coordinators are also responsible for foreign language translations and compiling a
       <a class="mw-redirect" href="/wiki/Bible_(writing)" title="Bible (writing)">
        show bible
       </a>
       that tracks plot points and character introductions as a reference tool.
      </p>
      <p>
       A script coordinator is a distinct role from
       <a href="/wiki/Script_supervisor" title="Script supervisor">
        script supervisor
       </a>
       . A script supervisor is responsible for on-set annotation of the
       <a href="/wiki/Screenplay" title="Screenplay">
        script
       </a>
       for editing purposes rather than
       <a href="/wiki/Pre-production" title="Pre-production">
        pre-production
       </a>
       coordination of the
       <a href="/wiki/Screenplay" title="Screenplay">
        script
       </a>
       to facilitate production. A script coordinator is considered a junior role in the writing staff and becoming a full-time
       <a href="/wiki/Screenwriter" title="Screenwriter">
        screenwriter
       </a>
       is typically a common promotion from being a script coordinator.
      </p>
